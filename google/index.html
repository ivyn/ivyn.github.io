Context

Sona Dolasia and I were project partners in Computer Science 188. When we worked online, we had trouble with equally contributing and understanding the material. We tried Google Docs, but had some pain points that we wanted to address by imagining what a collaborative code editor might look like in the Google Suite.


Research

Using our own experiences collaboratively coding, we brainstormed a list of features and did a competitive analysis on existing products.


In order to learn more about the role of collaboration in CS education, we compiled a question guide for both students and educators.


The insights from the interviews informed the direction of our project as well as our How Might We questions.



How Might We...

collaborate: make a seamless platform for real-time code collaboration
evolve: expand the Google Suite to encompass more tools for modern day tasks
educate: improve computer science education through collaboration
